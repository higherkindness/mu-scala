# freestyle-rpc

Simple RPC with Freestyle

## Installation

Add the following dependency to your project's build file.

For Scala 2.11.x and 2.12.x:

[comment]: # (Start Replace)

```scala
libraryDependencies += "io.frees" %% "frees-rpc" % "0.1.1"
```

[comment]: # (End Replace)
